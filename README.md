# mips32-cpp_emulator
A MIPS32 interpreter emulator in C++ using object oriented programming paradigm.

Supported instructions :
- One Parameter Instructions:
  - `mfhi`
  - `mflo`
  - `mthi`
  - `mtlo`
- Twp Parameter Instructions:
  - `la`
  - `li`
  - `move`
- Three Parameter Instructions:
  - `add`
  - `addi`
  - `sub`
  - `subi`
  - `mul`
  - `div`
